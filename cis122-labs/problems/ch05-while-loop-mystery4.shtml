Exercise : while loop mystery

Fill in the boxes at right with the output produced by each method call. 

public static void mystery4(int n) {
    int x = 1;
    int y = 2;
    while (y < n) {
        if (n % y == 0) {
            n = n / y;
            x++;
        } else {
            y++;
        }
    }
    System.out.println(x + " " + n);
}
mystery4(2); 
1 2
mystery4(5); 
1 5
mystery4(24); 
4 3
mystery4(28); 
3 7
