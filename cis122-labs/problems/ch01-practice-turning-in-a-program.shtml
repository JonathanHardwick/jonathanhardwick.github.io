Exercise : Practice turning in a program

In CSE 142 you'll use a web turnin system for your homework. Let's practice turning in a file by submitting your MyFirstProgram.java.

  • Go to the CSE 142 Labs web page.
  • Click the link that says "Turnin Page for turnin exercise."
  • Log in to the page and input your personal information.
  • Browse to your MyFirstProgram.java file and submit it.
  • Look at the turnin receipt page. Does it give you any warnings or errors? If so, fix them and try again. (A common error is giving a program/file the wrong name.)
