Exercise : What's the output?

  • How many lines of output are produced (including blank lines)? 
    public class Tricky {
    public static void main(String[] args) {
        System.out.println("Testing, testing,");
        System.out.println("one two three.");
        System.out.println();
        
        
        System.out.println("How much output");
        System.out.println();
        System.out.println("will there be?");
    }
}
  • Answer: 6 lines. The blank lines in the code don't count, but the System.out.println(); statements do.
