Exercise : Practice the Indenter Tool

In CSE 142 you can use a web page called the Indenter Tool to automatically correct the indentation of a program.

  • Go to the CSE 142 Labs web page.
  • Click the link that says "Indenter Tool."
  • Go back to your program. Select all of the code and copy/paste the code into the text box of the web page.
  • Click Fix Indendation on the page. The code should now be indented correctly. Copy/paste it back into jGRASP and save the file.
