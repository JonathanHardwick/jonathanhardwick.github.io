Exercise - jGRASP Debugger

  • Now you can debug the program by clicking on the debug icon instead of the run icon (it is to the right of the run icon and looks like a ladybug). It will stop every time it hits line 5. Then you can examine the values of the local variables i and number. When you want it to continue, you can hit the button that looks like the standard "play" icon of a right-pointing arrow.
