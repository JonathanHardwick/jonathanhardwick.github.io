Exercise : while loop mystery

Fill in the boxes at right with the output produced by each method call. 

public static void mystery(int x) {
    int y = 1;
    int z = 0;
    while (2 * y <= x) {
        y = y * 2;
        z++;
    }
    System.out.println(y + " " + z);
}
mystery(1); 
1 0
mystery(6); 
4 2
mystery(19); 
16 4
mystery(39); 
32 5
mystery(74);
