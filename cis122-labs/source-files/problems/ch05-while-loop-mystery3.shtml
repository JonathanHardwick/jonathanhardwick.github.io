Exercise : while loop mystery

Fill in the boxes at right with the output produced by each method call. 

public static void mystery3(int x) {
    int y = 0;
    while (x % 2 == 0) {
        y++;
        x = x / 2;
    }
    System.out.println(x + " " + y);
}
mystery3(19);

mystery3(42);

mystery3(48);

mystery3(40);
