Exercise : while loop mystery

Fill in the boxes at right with the output produced by each method call. 

public static void mystery2(int x, int y) {
    int z = 0;
    while (x % y != 0) {
        x = x / y;
        z++;
        System.out.print(x + ", ");
    }

    System.out.println(z);
}
mystery2(25, 2); 
12, 1
mystery2(32, 4); 
0
mystery2(10345, 10); 
1034, 103, 10, 3
mystery2(63, 2); 
31, 15, 7, 3, 1, 0, 6
